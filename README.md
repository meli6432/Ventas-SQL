# Proyecto: Análisis de Ventas con SQL y Power BI

## Objetivo
Analizar el comportamiento de las ventas, los clientes y los productos a partir de una base de datos relacional, con el fin de obtener indicadores clave de desempeño (KPIs) y generar un dashboard interactivo que apoye la toma de decisiones comerciales.

## Herramientas utilizadas
- SQL: creación de tablas y consultas de análisis
- Python (Jupyter Notebook) en Visual Studio Code: ejecución de consultas SQL, exploración de datos y exportación a CSV
- Pandas: manipulación y visualización tabular de los resultados
- Power BI: creación del dashboard interactivo
- GitHub: publicación y documentación del proyecto

## Proceso
1. Inserción de datos simulados con un enfoque realista.

2. Desarrollo de consultas SQL para:
- métricas generales de ventas
- segmentación por ciudad
- análisis de productos y clientes
- evolución temporal de ingresos

3. Exportación de las tablas a archivos CSV desde Jupyter Notebook.

4. Importación de los datos en Power BI y modelado de relaciones.

5. Creación de medidas DAX para los principales KPIs.

6. Diseño y construcción del dashboard final.


## Visualización de datos
El dashboard en Power BI incluye:
- KPIs principales: ingresos totales, número de ventas, clientes únicos y ticket promedio.
- Gráfico de ventas por mes para identificar tendencias.
- Ingresos por ciudad para analizar el desempeño geográfico.
- Productos más vendidos y productos con mayores ingresos.
- Ranking de clientes con mayor gasto.
- Filtros interactivos por ciudad, categoría y fecha


## Resultados / Conclusiones
- Se identificaron las ciudades con mayor generación de ingresos, lo que permite priorizar esfuerzos comerciales.

- Se observó que los productos más vendidos no siempre son los que generan mayores ingresos, destacando la importancia del análisis de rentabilidad.

- El análisis temporal permitió detectar patrones de ventas y posibles picos de demanda.

- El dashboard facilita el seguimiento de métricas clave y permite explorar los datos de forma interactiva para apoyar la toma de decisiones.
